Quiz API v1.0.0

Simple quiz generation for your applications.

POST /quiz

Generates a list of questions based on your specific criteria.

Request Parameters

Parameter Type Description
number Integer The total number of questions to return.
topic String The subject matter (e.g., "Science", "History").
difficulty String Level of challenge: easy, medium, or hard.

Response Format

Returns a JSON object where keys are questions and values are the correct answers. 

{
  "What is the capital of France?": "Paris",
  "Who wrote 'Romeo and Juliet'?": "William Shakespeare"
}
